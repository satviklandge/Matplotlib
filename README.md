# Matplotlib — Data Visualization with Python

> A hands-on collection of Jupyter notebooks covering the most essential Matplotlib chart types, built around real-world data.

---

## Overview

This repository serves as a practical guide to **Matplotlib**, Python's most popular data visualization library. Each notebook is self-contained, focusing on a single chart type with clear examples and customization tips — perfect for beginners getting started with data visualization or analysts looking for quick reference material.

The notebooks use the **Google Play Store dataset** (`googleplaystore.csv`) to ground examples in real data, making the visualizations immediately relevant and meaningful.

---

## Repository Structure

| # | Notebook | Description |
|---|----------|-------------|
| 01 | `01_Matplotlib_lineplot.ipynb` | Plotting trends over time using line charts |
| 02 | `02_Matplotlib Histogram.ipynb` | Understanding data distributions with histograms |
| 03 | `03_Matplotlib Scatter Plot.ipynb` | Exploring relationships between variables |
| 04 | `04_Matplotlib Barplot.ipynb` | Comparing categories with bar charts |
| 05 | `05_Matplotlib Pieplot.ipynb` | Visualizing proportions with pie charts |
| 06 | `06_Matplotlib Box plot.ipynb` | Analyzing spread and outliers with box plots |
| 07 | `07_Matplotlib Subplot.ipynb` | Combining multiple plots in a single figure |

**Dataset:** `googleplaystore.csv` — used across notebooks for real-world examples.

## Getting Started

### Prerequisites

Make sure you have Python 3.x installed along with the following libraries:

```bash
pip install matplotlib pandas numpy jupyter
```

### Running the Notebooks

1. **Clone the repository**
   ```bash
   git clone https://github.com/satviklandge/Matplotlib.git
   cd Matplotlib
   ```

2. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

3. **Open any notebook** from the file browser and run the cells sequentially.

## What You'll Learn

- How to create and customize **line plots** for trend analysis
- Building **histograms** to visualize frequency distributions
- Using **scatter plots** to identify correlations and patterns
- Crafting **bar charts** for category-level comparisons
- Designing **pie charts** to show part-to-whole relationships
- Interpreting data spread and outliers with **box plots**
- Organizing multiple visualizations using **subplots**
- Applying labels, titles, legends, colors, and styling across all chart types
